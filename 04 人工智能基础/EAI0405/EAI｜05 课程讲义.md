# EAI｜05 模型导出机制与 Web 前端部署基础

课程名称：人工智能基础 

单元主题：AI模型前端部署与网页交互实现 

授课年级：初中高年级 / 高一 

计划课时：90分钟

## 课程概括

本节课聚焦于Teachable Machine模型导出与网页前端集成过程，帮助学生理解模型结构文件的组成、部署逻辑及在HTML网页中进行动态识别的技术路径。通过构建一个图像识别网页Demo，学生将亲手实现模型加载、推理结果获取与页面响应的完整流程，初步掌握TensorFlow.js部署的基本方法。

## 教学目标

### 🎯知识与技能

- 了解TensorFlow.js模型的结构组成（.json配置与.bin权重）；
- 掌握Teachable Machine模型的导出与引用方法；
- 能够编写基础的HTML与JavaScript用于模型加载与结果显示；
- 实现图像识别触发网页元素变化的基础交互效果。

### 🎯过程与方法

- 演示模型导出后的结构组成与作用；
- 讲解模型部署在前端网页中的加载机制；
- 学生分组实践加载模型并实现前端页面响应（如背景色改变）；
- 观察模型响应速度与错误反馈机制，尝试基本调试。

### 🎯情感态度价值观

- 鼓励学生体验从模型训练到系统部署的完整流程；
- 培养解决实际工程问题的技术耐心与动手能力；
- 强化“技术落地”视角，理解模型不仅是“训练好”，更要“跑得通”。
